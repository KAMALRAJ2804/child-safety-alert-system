# Child Safety Alert System (CSAS)

## Overview
This project is designed to detect if a child is left inside a parked vehicle and send alerts to prevent accidents.

## Objective
To improve child safety by monitoring vehicle interiors using sensor-based detection.

## Components Used
- ESP32 Microcontroller
- FSR Sensor
- PIR Sensor
- Wi-Fi Module

## Working Principle
- The FSR sensor detects weight/pressure
- The PIR sensor detects motion
- If both conditions are met, the system triggers an alert
- Notification is sent via Wi-Fi

## Features
- Real-time monitoring
- Dual-sensor accuracy
- Wireless alert system
